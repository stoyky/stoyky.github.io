## Block notation

Alternatively, block notation can be generated using `$$` delimiters. This will output the expression in its own HTML block.

**Example:**

```tex
% KaTeX block notation
$$
 \varphi = 1+\frac{1} {1+\frac{1} {1+\frac{1} {1+\cdots} } }
$$
```
